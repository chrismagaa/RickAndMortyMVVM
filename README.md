# Rick And Morty MVVM
App to consume The Rick and Morty API in kotlin.

## Description

* Tech/Tools
   - Kotlin
   - Coroutines
   - LiveData and ViewModel
   - Hilt
   - Coil 
   - Retrofit
   - Paging 3

* Architecture
   - MVVM
